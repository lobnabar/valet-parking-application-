# Valet Parking Application - Manual Testing

This repository contains the manual testing work for the **Valet Parking Application** project.  
The testing process was organized and tracked using Trello.

## Testing Scope

- **UI Testing**  
  Verified the user interface elements and workflows.

- **Valid Test Cases**  
  Designed scenarios with correct inputs and expected outcomes.

- **Invalid Test Cases**  
  Covered scenarios with incorrect inputs to check error handling.

- **Corner Cases**  
  Tested edge cases to ensure application stability.

- **Bug Reports**  
  Documented issues found during testing with clear reproduction steps.

## Purpose

The goal of this project is to ensure the Valet Parking Application meets quality standards through structured manual testing and detailed bug reporting.
